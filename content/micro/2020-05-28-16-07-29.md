TIL about `uuidgen` (on Mac):

    NAME
        uuidgen -- generates new UUID strings

    SYNOPSIS
        uuidgen [-hdr]

    DESCRIPTION
        The uuidgen command generates a Universally Unique IDentifier (UUID), a 128-bit value guaranteed to be unique over both space
        and time.
