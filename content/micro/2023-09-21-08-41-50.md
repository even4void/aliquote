Thanks to [notify-send.sh](https://github.com/vlevit/notify-send.sh) I can now update the mail notification from Neomutt in Ubuntu notification center since it allows to replace an existing notification. This should be available in more [recent versions](https://askubuntu.com/q/161851) of `notify-send` (Ubuntu â‰¥ 22.10).

```
set new_mail_command="notify-send.sh --replace-file=/tmp/muttnotification -a neomutt -u low -i neomutt 'New Email' '%n new messages, %u unread.' &"
```

Beforehand I tried [notify-send.py](https://github.com/phuhl/notify-send.py) which didn't quite work.
